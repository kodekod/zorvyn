# Financial Dashboard (Frontend Only)

A modern and responsive **Financial Dashboard UI** built using **React** and **Tailwind CSS**.
This project focuses on creating a clean, interactive, and user-friendly interface for visualizing financial data.

---

##  Live Demo

link 
https://vercel.com/sekhma/zorvyn

##  Project Overview

This is a **frontend-only dashboard project** designed to showcase modern UI development skills.
It includes financial summary cards, charts, transaction tables, and interactive features like search, filtering, and sorting.

The goal of this project is to build a professional dashboard similar to real-world financial and admin panel applications.

---

##  Features

Dashboard Summary Cards (Balance, Income, Expenses)
Interactive Line Chart for financial trends
Pie Chart for category-wise breakdown
Transactions Table
Search functionality
Filter options
Sorting functionality
Role-based UI (Admin / Viewer)
Fully responsive design
Reusable components

---

##  Tech Stack

* React.js
* Tailwind CSS
* JavaScript
* Chart Library (Recharts / Chart.js)
